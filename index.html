QSN 3: Methods of requesting resource
  1. Get method requests a representation of the specified resource. Requests using this method should only retrieve data.
  2. Head asks for a response identical to that of a get request, but without the response body.
  3. Delete method deletes a specific request
  4. Connect establishes a tunnel to the server identified by the target resource
